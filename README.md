# GoPose
GoPose是人体姿态关键点自动识别、编辑软件，同时可作为运动学分析软件  
## Background
那年做毕业论文，用北体的**视讯**软件，半年时间逐帧逐点手动标记运动员的关键点，最终顺利完成论文，非常感谢视讯  
但手动标点工作会耗费大量时间、使人疲惫，“如果有自动识别关键点的功能该多好”的想法那时诞生  
最近闲，实现一下当年的想法
